# Pages
A reader's book log - a place to track, review, and share what you read.
## Overview 
Pages is a social platform for readers to connect with others through their reading interests.
- ## Tech Stack
- **Framework:** Next.js (App Router)
- **Database:** Prisma + PostgreSQL
- **Auth:** NextAuth.js
- **Styling:** TailwindCSS
- **Hosting:** Vercel (planned)
